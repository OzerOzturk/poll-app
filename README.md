## How to run this app on your local machine?

1. Clone the repo
   ```sh
   git clone https://github.com/OzerOzturk/poll-app.git
   ```
2. Navigate to the project folder

   ```sh
   cd poll-app
   ```

3. Install dependencies
   ```sh
   npm install
   ```
4. Run the project
   ```sh
   npm start
   ```
